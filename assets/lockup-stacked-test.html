Use Case Recommendations

STACKED WORKS BEST FOR
  • App icons / profile pictures
  • Square format social media
  • Centered hero sections
  • Formal / institutional feel
  • Print materials
HORIZONTAL WORKS BEST FOR
  • Navigation headers
  • Wide format banners
  • Left-aligned layouts
  • Modern / tech feel
  • Small sizes (saves height)
