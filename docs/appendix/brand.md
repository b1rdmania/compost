## Usage Examples

### Do

::: tip Correct Usage
- Use logo on dark backgrounds
- Maintain clear space
- Use brand green for CTAs and links
- Use earthy tones for secondary elements
:::

### Don't

::: danger Avoid
- Don't place logo on busy backgrounds
- Don't stretch or distort the logo
- Don't use off-brand colors
- Don't use light mode without adjustment
:::
