# Institutional Access

For funds, family offices, and platforms requiring compliance, reporting, and dedicated support.

---

## Onboarding

| Step | Description |
|------|-------------|
| KYC/AML verification | Through regulated partner |
| Investor qualification | Accredited/qualified purchaser |
| Legal entity contracting | Subscription agreement |
| Risk disclosures | Review and acknowledge |

---

## Reporting

| Report | Frequency |

## Advanced Structures

### Segregated Vaults

For larger allocations or compliance requirements:

| Feature | Description |
|---------|-------------|
| Separate vault instance | Not commingled with retail |
| Jurisdiction-specific | Where needed |
| Own receipt token | cHYPE-[X] |
| Same strategy | Same allocation, same yield |

### White-Label

Offer Compost yield to your clients:

| Feature | Description |
|---------|-------------|
| Your brand | Your UI, your client relationships |
| API access | Full vault infrastructure |
| Custom fees | Negotiated arrangements |

### Feeder Funds

Regulated vehicle (Cayman, BVI, etc.) that invests into Compost:

| Benefit | Description |
|---------|-------------|
| Clean structure | For LPs with fund mandates |
| Admin handled | Fund service providers |
| Audit included | Third-party verification |
